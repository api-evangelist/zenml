# ZenML (zenml)
ZenML is a unified MLOps framework that enables teams to develop, evaluate, and deploy machine learning pipelines and AI agents on any infrastructure. It provides a Python SDK, REST API, and server for managing ML pipelines, stacks, artifacts, and deployments across cloud providers.

## APIs

### ZenML API
The ZenML REST API provides programmatic access to manage ML pipelines, stacks, artifacts, models, and deployments in ZenML. It is used by the Python SDK and can be accessed directly for automation and integration with CI/CD systems and MLOps workflows.
